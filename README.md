# GradeMaster Python: UBIT GPA Manager 🎓

A command-line interface (CLI) tool designed for students at the University of Karachi (UBIT) to track academic progress and calculate GPAs.

## ✨ Features
- **Persistent Storage:** Uses JSON to save data so your grades aren't lost when you close the program.
- **Dynamic Calculation:** Automatically calculates weighted GPA based on user input.
- **Error Handling:** Validates inputs to ensure only numerical grades are processed.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Data Format:** JSON (JavaScript Object Notation)
- **Libraries:** `os`, `json`

## 🚀 How to Run
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/UBIT-GPA-Manager.git`
2. Run the script: `python gpa_manager.py`
